# Client 1

Application React avec Tailwind CSS pour l'interface utilisateur.

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

## Production

```bash
npm run build
```

## Docker

Pour construire l'image Docker:

```bash
docker build -t client1 .
```

Pour exécuter le conteneur:

```bash
docker run -p 3000:3000 client1
```
