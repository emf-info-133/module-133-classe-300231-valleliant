# Client 2

Application React avec Tailwind CSS pour l'interface utilisateur.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ou alternativement:

```bash
npm start
```

## Production

```bash
npm run build
```

## Docker

Pour construire l'image Docker:

```bash
docker build -t client2 .
```

Pour exécuter le conteneur:

```bash
docker run -p 3001:3000 client2
```
